# Linux Dotfiles

This repository contains my configuration files for Linux.

## Usage

To use a specific theme, navigate to its folder and follow the setup instructions (if any).

> Each folder represents a different theme or setup.

## Themes

- `catppuccin-mocha-lavender` – based on [catppuccin](https://catppuccin.com/palette/) **mocha** color palette, uses **lavender** as accent color.
